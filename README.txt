IP Camera Details Checker mk.1

This application fetches and scans the detail of an IP Camera. 
It returns Serial numbers and MAC Address by accessing them through the browser. 
This supports both Revotech and Hikvision cameras.

Required OS: Windows
Required Application: Chrome Browser

HOW TO USE:

1. Run the "camera.py" file in this folder
2. Type the IP Address of a camera that you want to scan
3. This application automatically use your Chrome browser to open and access the IP Camera.
4. Wait for the results

ENJOY!
